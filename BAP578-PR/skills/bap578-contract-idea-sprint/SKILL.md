# BAP578 Contract Idea Sprint

## Usage

- Category: BAP578 Dev
- Mode: guide
- Version: 0.1.0

## Input Example

```json
{
  "idea": "Agent escrow for on-chain game rewards"
}
```

## Output Example

```text
- Define ownership and operator boundaries
- Draft adapter contract interface and storage
- Plan one-day implementation/testing milestones
```

## Security Notes

- Include threat assumptions for each feature idea before coding.
- Add auth/balance checks to the first test plan draft.

## Install

`npx @skillshub/bap578-contract-idea-sprint`
